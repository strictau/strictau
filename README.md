## strictau

I build esports platforms and the tools that run them, mostly web apps and Discord bots.

**[Esports Hub](https://esportshub.io)** is the all-in-one esports career platform: networking, recruitment, verified mentors, contracts, custom Discord bots, and team sites in one place.

**[Wave Network](https://wavenet.gg)** is the network behind modern esports, the marketing and content hub for the org and its divisions, with a blog and podcast library.
